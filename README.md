## Getting Started


### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```


## Installation
1) clone the project
```bash
 git clone https://github.com/kartikpavan/User_Auth.git
```
2) install the necessary dependencies
``` bash
npm install
```
3) start the development server
``` bash
npm run start 
```
